# Nick's Mods — Site

Static single-page site for Nick's Mods (retro handheld repair, Oviedo FL).

## Structure
- `index.html` — the whole site (HTML/CSS/JS in one file)

## Workflow
1. Edit `index.html`
2. `git add -A`
3. `git commit -m "describe the change"`
4. `git push`

## Deploy
Hosted via GitHub Pages once the repo is pushed to GitHub — see setup steps in commit history / project notes.
